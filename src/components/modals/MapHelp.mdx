#### Map Controls

| Icon | Click action |
|------|-------------|
| <NorthArrowIcon size={24} /> | Align the map to true north |
| <Icon name="fa7-solid:plus" size={18} /> | Zoom in |
| <Icon name="fa7-solid:minus" size={18} /> | Zoom out |
| <ResetToSearchCenterIcon class="help-icon" /> | Reset the view to the search center markers |
| <Icon name="fa7-solid:circle-arrow-down" size={24} /> | Look straight down |
| <img src={layersIcon.src} /> | Select base map layer |
| <HelpIcon size={24} /> | Map help |

#### Bottom Panel Map Controls

| Icon | Click action |
|------|-------------|
| <Icon name="fa7-solid:xmark" size={24} /> | Exit the active track and go back to world view|
| <Icon name="fa7-solid:crosshairs" size={24} /> | Reset the view to the active track |
